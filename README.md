# 任务管理系统

## 项目简介

这是一个基于 Vue 3 开发的任务管理系统，支持用户个性化管理任务，具备添加、编辑、删除、状态切换、筛选与统计等核心功能。为本人期末大作业

## 技术栈

- **前端框架**：Vue 3 (Composition API + `<script setup>`)
- **路由管理**：Vue Router 4
- **UI 设计**：HTML5 + CSS3 + JavaScript
- **数据存储**：localStorage
- **构建工具**：Vite
- **响应式布局**：Flex + Grid

## 核心功能

### 1. 任务管理
- ✅ 任务添加：支持标题、描述、优先级、标签和截止日期
- ✅ 任务编辑：支持编辑标题、描述、优先级和日期
- ✅ 任务删除：带确认模态框的安全删除
- ✅ 状态切换：支持待办/进行中/已完成
- ✅ 优先级管理：高/中/低三级优先级，颜色区分
- ✅ 标签功能：支持添加多个标签，便于分类管理

### 2. 搜索与排序
- ✅ 全局搜索：支持按标题和描述搜索任务
- ✅ 多维度排序：支持按截止日期、创建时间、优先级排序
- ✅ 升序/降序切换：点击排序按钮切换顺序

### 3. 批量操作
- ✅ 多选功能：支持勾选单个或所有任务
- ✅ 批量删除：一键删除多个任务
- ✅ 批量完成：一键标记多个任务为已完成

### 4. 数据统计
- ✅ 实时统计：显示任务总数、待办/进行中/已完成百分比
- ✅ 数据可视化：统计卡片展示，支持动画效果
- ✅ 响应式图表：适配不同屏幕尺寸

### 5. 日历功能
- ✅ 月视图日历：显示每月任务分布
- ✅ 日期跳转：支持快速切换月份
- ✅ 任务标记：在对应日期显示任务数量

### 6. 深色模式
- ✅ 手动切换：支持深色/浅色模式切换
- ✅ 系统适配：自动检测系统主题偏好
- ✅ 主题记忆：本地存储主题偏好
- ✅ 平滑过渡：主题切换动画效果

### 7. 响应式设计
- ✅ PC端适配：≥1920×1080
- ✅ 平板适配：768×1024
- ✅ 移动端适配：375×667
- ✅ 流畅的交互体验：动画过渡效果
- ✅ 无障碍访问：合理的颜色对比度

### 8. 动画效果
- ✅ 元素淡入：页面加载时的渐入动画
- ✅ 悬停效果：卡片和按钮的交互反馈
- ✅ 状态过渡：任务状态变化的平滑过渡
- ✅ 弹窗动画：模态框的缩放效果
- ✅ 数值动画：统计数据的增长动画

```
│   └── TaskStatsPanel.vue    # 统计面板
├── views/           # 页面级组件
│   ├── HomeView.vue          # 首页 - 任务列表与操作区
│   ├── StatsView.vue         # 统计页 - 可视化任务分布

├── utils/           # 工具函数

├── App.vue          # 根组件 - 路由管理与主题切换

```

## 组件说明

### Calendar.vue
- **功能**：日历视图组件，用于展示任务日期分布
- **交互**：
  - 点击月份导航切换月份
  - 对应日期显示任务数量标记
  - 支持快速跳转到特定月份
- **数据**：接收任务列表，按日期分组显示
- **特性**：响应式设计，适配不同屏幕尺寸

### TaskInput.vue
- **功能**：任务输入表单，支持创建新任务
- **交互**：
  - 输入标题和描述
  - 选择优先级和截止日期
  - 输入标签（逗号分隔）
  - 点击添加按钮创建任务
- **事件**：`task-added` - 任务添加成功时触发
- **特性**：表单验证，自动聚焦，输入提示

### TaskList.vue
- **功能**：任务列表展示与管理
- **交互**：
  - 搜索框实时过滤任务
  - 排序选择器切换排序方式
  - 状态筛选按钮筛选任务
  - 任务状态快速切换
  - 双击编辑任务标题和描述
  - 点击编辑优先级和日期
  - 批量选择和操作
  - 带确认的删除功能
- **事件**：`task-updated` - 任务更新时触发，`task-deleted` - 任务删除时触发
- **特性**：响应式设计，动画过渡，空状态提示

### TaskStatsPanel.vue
- **功能**：任务统计面板，展示任务数据
- **数据**：
  - 总任务数
  - 待办任务百分比
  - 进行中任务百分比
  - 已完成任务百分比
- **特性**：
  - 数值增长动画
  - 悬停效果
  - 响应式布局
  - 渐变背景色
  - 12列栅格系统

## 数据结构

### 任务对象

```javascript
{
  id: string,           // 唯一标识
  title: string,        // 任务标题
  description: string,  // 任务描述
  status: string,       // 状态：todo/in-progress/done
  priority: string,     // 优先级：high/medium/low
  dueDate: string,      // 截止日期
  tags: Array<string>,  // 标签数组
  createdAt: string,    // 创建时间
  updatedAt: string     // 更新时间
}
```

### 优先级枚举

```javascript
{
  HIGH: 'high',     // 高优先级 - 红色
  MEDIUM: 'medium',  // 中优先级 - 橙色
  LOW: 'low'         // 低优先级 - 绿色
}
```

### 状态枚举

```javascript
{
  TODO: 'todo',           // 待办
  IN_PROGRESS: 'in-progress',  // 进行中
  DONE: 'done'           // 已完成
}
```
